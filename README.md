# 2026_multib_paper_AlfaPLUS

MATLAB scripts used in the analyses for the manuscript:
Kassinopoulos et al. (2026). Intracellular fluid accumulation underlies brain volume increases in early Alzheimer’s disease. Brain Communications.

The full manuscript, including methodological details, is available through Brain Communications.

# Overview
This repository contains the MATLAB scripts used to reproduce the analyses presented in the manuscript.
The code supports:
- voxelwise statistical analyses (SPM-based and custom MATLAB),
- cluster‑level extractions, and
- figure generation.

# Requirements
- MATLAB R2023a (or later recommended)
- SPM12
- "Tools for NIfTI and ANALYZE image" library from MATLAB File Exchange (link: https://uk.mathworks.com/matlabcentral/fileexchange/8797-tools-for-nifti-and-analyze-image)
